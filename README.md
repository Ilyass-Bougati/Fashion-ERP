# Fashion ERP
Fashion-ERP is an innovative, web-based Enterprise Resource Planning (ERP) system tailored for the fashion industry. Engineered with modern web technologies and augmented by Artificial Intelligence, this system aims to streamline business operations, provide intelligent sales forecasting, and automate financial reporting.

This project is developed as part of a Software Engineering & AI academic curriculum.

## Scope
- *Security & User Management*: Role-Based Access Control (RBAC) to ensure secure and appropriate data access.
- *Analytics & Dashboards*: Comprehensive statistical dashboards displaying quarterly, monthly, and annual metrics.
- *AI-Powered Sales Forecasting*: Predictive analytics utilizing AI to anticipate sales trends.
- *Employee Management*: Tracking of sales performances and automated commission calculations.
- *Dynamic LLM Reporting*: Integration of Large Language Models (LLMs) to automatically generate Audit, Finance, and Tax reports.
- *Financial Centralization*: Streamlining financial operations using Webhooks.
- *Inventory Management*: Robust stock tracking and management.

## Tech Stack
- *Backend*: Spring Boot (Java)
- *Frontend*: Next.js (React)
